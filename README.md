# MiMo AI Study Assistant

This is a lightweight AI assistant project based on Xiaomi MiMo API.

## 🚀 Features

- 📄 Academic paper summarization (EN → CN)
- 🧠 Course content explanation and structured notes
- 💡 Research idea generation
- 💻 Coding assistance (debugging & explanation)

## 🧪 Use Cases

- University students learning AI / EE / CS
- Paper reading and knowledge extraction
- Quick project prototyping with LLM

## 🛠 Tech Stack

- Python / Node.js (planned)
- Xiaomi MiMo API
- Prompt Engineering

## 📌 Status

Currently in early prototype stage.

## 📷 Demo (planned)

- CLI-based interaction
- Web demo (coming soon)

## 🔗 API Usage

This project is used to test:

- Long context understanding
- Chinese academic writing
- Code reasoning ability

## 📎 Notes

This repository is mainly used for experimentation and evaluation of MiMo model capabilities.
